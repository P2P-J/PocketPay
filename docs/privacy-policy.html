개인정보처리방침

작은 모임(이하 "서비스")은 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을 준수합니다.

1. 수집하는 개인정보 항목

회원가입 시

  • 이메일 주소, 닉네임, 비밀번호(암호화 저장)

소셜 로그인 시

  • 소셜 계정 고유 ID, 닉네임, 이메일(제공 시)
  • 지원 플랫폼: 카카오, 네이버, 구글, Apple

서비스 이용 시

  • 거래 내역 정보 (상점명, 금액, 날짜, 카테고리, 설명)
  • 모임 정보 (모임명, 설명, 멤버 목록)
  • 영수증 이미지 (OCR 처리 후 즉시 삭제) — 사용자가 카메라 또는 사진 라이브러리에서 직접 선택한 이미지에 한함

기기 권한 사용

  • 카메라(android.permission.CAMERA): 영수증 직접 촬영 후 OCR 자동 입력에만 사용. 이미지는 OCR 처리 즉시 서버에서 삭제되며 별도 저장하지 않음
  • 사진 라이브러리: 기존 영수증 이미지를 선택하여 OCR 처리에 사용

자동 수집

  • 기기 정보, 접속 로그, 앱 사용 기록

2. 개인정보의 수집 및 이용 목적

  • 회원 식별 및 서비스 제공
  • 소모임 공금 관리 기능 제공 (거래 기록, 정산, 통계)
  • 서비스 개선 및 오류 해결
  • 고객 문의 대응

3. 개인정보의 보유 및 이용 기간

  • 회원 탈퇴 시까지 보유
  • 탈퇴 시 즉시 파기 (단, 법령에 의한 보관 의무가 있는 경우 해당 기간 동안 보관)
  • 전자상거래법에 따른 거래 기록: 5년

4. 개인정보의 제3자 제공

  • 원칙적으로 제3자에게 제공하지 않습니다.
  • 법령에 의한 요청이 있는 경우 예외

5. 개인정보의 파기

  • 수집 목적이 달성된 후 즉시 파기
  • 전자적 파일: 복구 불가능한 방법으로 삭제
  • 영수증 이미지: OCR 처리 즉시 서버에서 삭제

6. 이용자의 권리

  • 개인정보 열람, 정정, 삭제, 처리 정지 요청 가능
  • 회원 탈퇴를 통해 모든 개인정보 즉시 삭제 가능
  • 앱 내 "더보기 > 회원 탈퇴"에서 직접 처리 가능

7. 개인정보의 안전성 확보 조치

  • 비밀번호 암호화 저장 (bcrypt)
  • 통신 구간 암호화 (HTTPS/TLS)
  • 인증 토큰 안전 저장 (iOS Keychain / Android EncryptedSharedPreferences)
  • 접근 권한 최소화 및 API Rate Limiting
